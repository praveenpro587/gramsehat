# GramSehat Demo Cases 🎬
# Use these during your hackathon demo for maximum impact

## Case 1 — RED EMERGENCY (Best for demo climax)
- **Patient:** Ramabai Patil, Age 6, Maharashtra, Monsoon
- **Language:** Marathi
- **Symptoms:** "3 दिनों से तेज बुखार है, कंपकंपी आती है, आंखें पीली दिख रही हैं, खाना नहीं खा रही"
  (Translation: High fever for 3 days, chills, yellow eyes, not eating)
- **Expected Diagnosis:** Malaria with possible Jaundice → RED 🔴
- **Demo Impact:** Shows emergency alert + ASHA worker notification

## Demo Flow Script (3 Minutes)

### Minute 1 — The Problem (Emotional Hook)
"Every day, 800 million rural Indians have almost no access to doctors.
A sick child in Vidarbha could be 50km from the nearest clinic.
By the time they travel, it might be too late.
We built GramSehat to put an AI doctor in every village."

### Minute 2 — Live Demo (Use Case 1)
1. Enter Ramabai, Age 6, Marathi, Maharashtra, Monsoon
2. Type the Hindi symptoms
3. Click Analyze → show Marathi response
4. Upload a jaundiced eye photo (stock image)
5. Click Analyze Image → show visual findings
6. Click Get Diagnosis → RED alert appears dramatically
7. Show WhatsApp notification going to ASHA worker

### Minute 3 — Impact & Scale
"This plugs into India's 1 million ASHA workers.
It uses Groq — fastest AI inference in the world — under 500ms responses.
It works on 2G. It costs less than ₹5 per patient.
This is healthcare equity at scale."
